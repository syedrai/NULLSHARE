## 🚀 Getting Started

### Step 1: Start Server
```bash
start.bat
```

### Step 2: Open Dashboard
```
http://localhost:3000
```

### Step 3: Create Share
1. Enter folder path
2. Set label
3. Click "Generate Share Link"

### Step 4: Access Share
1. Open share URL
2. Enter password (if set)
3. Browse and download files

### Step 5: Monitor Activity
1. Go to "Live Logs" tab
2. See real-time events
